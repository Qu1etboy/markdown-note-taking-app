# Markdown note taking app

Work in progress.

## Client

- react
- tailwindcss
- vitejs

## Server

- nodejs
- express
- mysql

## Setup

- clone this repo

```
git clone https://github.com/Qu1etboy/markdown-note-taking-app.git
```

- run server

```
cd server
npm run dev
```

- run client

```
cd client
npm run dev
```
